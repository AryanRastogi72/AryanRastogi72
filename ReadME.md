## About Me

I like understanding what's happening beneath the abstraction layer — a REST API call, a database transaction, a clock cycle inside an FPGA. That curiosity has taken me across digital hardware design, full-stack web development, and multi-agent AI systems, and I haven't picked a lane yet on purpose. Each project below reflects a different thing I was trying to learn at the time.

## Featured Projects

### 🎓 SARMS — University ERP System
Full-stack University ERP serving three role-based portals for Students, Faculty, and Admin. JWT-based stateless authentication with route guards, automated SGPA computation, grade finalization, and semester promotion logic.

**Stack:** React 19, Spring Boot 3, MongoDB, JWT, Tailwind CSS
**Team Project, 4 Contributors** · **[View Repository](https://github.com/top-of-the-code/SARMS)**

### 🔬 FSM-Based RFFT Spectrum Analyzer
Real-time N=16 FFT spectrum analyzer implemented in Verilog on a Xilinx Artix-7 FPGA. FSM-driven datapath with Q1.15 fixed-point arithmetic and live VGA bar-graph output at 640x480 @ 60Hz, synthesized and deployed on a Digilent Nexys 4.

**Stack:** Verilog, Xilinx Vivado, FPGA (Artix-7)
**Team Project, 4 Contributors** · **[View Repository](https://github.com/aniket-s007/FSM_based_RFFT)**

### 🏦 RANS Bank — Online Banking Management System
Full-featured banking platform with role-based access for Customers, Staff, and Admins. MySQL stored procedures handle loan lifecycles and EMI schedules, with row-level locking to prevent race conditions during concurrent fund transfers.

**Stack:** Python, Flask, MySQL, Jinja2
**Team Project, 4 Contributors** · **[View Repository](https://github.com/RaghavGarg3594/Online_Banking_Management_System)**

### ✈️ Multi-Agent AI Travel Planner
Multi-agent AI system using a Map-Reduce workflow to coordinate real-time flight and hotel data across 5+ API endpoints, with Amadeus and OpenAI APIs generating dynamic itinerary and pricing recommendations.

**Stack:** Python, LangGraph, Amadeus API, OpenAI API, SQLite
**[View Repository](https://github.com/AryanRastogi72)**
